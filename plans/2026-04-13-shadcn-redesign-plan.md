# Shadcn Redesign Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Restyle the entire SmilesDrawer documentation/marketing site using shadcn/ui primitives and shadcnblocks.com block patterns. Visual + UX scope only — no library swap, no relocation, no content rewrites. Single feature branch, merged once the whole site looks right.

**Architecture:** Astro 5 stays as the static-site framework. Add `@astrojs/react` integration so React islands can host shadcn primitives where interactivity is needed. Static pages remain `.astro` files using shadcn token classes. Four React islands: `HeroPlayground`, `PlaygroundApp`, `InstallTabs`, `ExamplesFilter`. Shadcn's CSS-variable token system is the single source of truth for colors, spacing, radii — used by both `.astro` markup and `.tsx` islands. Dark mode keeps its current implementation: the inline flash-free init script in `Layout.astro` head sets `class="dark"` on `<html>`, and shadcn reads that class. SmilesDrawer keeps loading from the existing local `public/js/smiles-drawer.min.js` script (already at v2.2.1) — **not** a CDN, not the npm package. The npm switch and any relocation are deferred to a separate future PR.

**Tech Stack:**
- Astro 5 (`^5.5.0`), `@astrojs/tailwind` (`^6.0.2`), Tailwind CSS 3 (`^3.4.17`)
- New: `@astrojs/react` integration, `react`, `react-dom`, type packages
- New: `shadcn@latest` CLI, primitives generated into `src/components/ui/`
- New: `lucide-react` (icons inside `.tsx` islands), `astro-icon` + `@iconify-json/lucide` (icons inside `.astro` files)
- Existing: SmilesDrawer v2.2.1 served from `public/js/smiles-drawer.min.js`
- Existing: Inter + JetBrains Mono from Google Fonts (currently loaded but mismatched in `tailwind.config.mjs` — this plan fixes that)

**Spec:** [`specs/2026-04-13-shadcn-redesign-design.md`](../specs/2026-04-13-shadcn-redesign-design.md)

**Verification model:** This project has no test runner. Verification for every task is manual: run `npm run dev` (kept running throughout the migration), open the relevant page in a browser, check light + dark mode + mobile + desktop. The final task is a full visual QA pass plus a production build verification.

```ts
export function cn(...inputs: ClassValue[]) {

/**
 * Resolve a path against the configured Astro base URL.
 *
 * Use this in every .tsx React island for asset URLs and internal links.
 * Static .astro files do not need this — they already use `import.meta.env.BASE_URL`
 * directly via the existing `const base = import.meta.env.BASE_URL` pattern.
 *
 * Example: withBase('/playground') → '/smiles-drawer-website/playground'
 */
export function withBase(path: string): string {
    const base = import.meta.env.BASE_URL || '/';
    const trimmedBase = base.endsWith('/') ? base.slice(0, -1) : base;
    const trimmedPath = path.startsWith('/') ? path : `/${path}`;
    return `${trimmedBase}${trimmedPath}`;
}
```

- [ ] **Step 2: Create `src/lib/use-smiles-drawer.ts`**

Create a new file `src/lib/use-smiles-drawer.ts` with these contents:

```ts
import {useEffect, useState} from "react"

/**
 * React hook that resolves once `window.SmilesDrawer` is defined.
 *
 * The local SmilesDrawer build is loaded via a script tag in Layout.astro and
 * may not be available the moment a React island mounts. This hook polls for
 * the global on a short interval and resolves with `{ ready: true, drawer }`
 * once it appears. Components should guard their rendering on `ready` and
 * call SmilesDrawer methods inside useEffect (rendering writes to the DOM).
 *
 * After ~5 seconds without finding the global, the hook reports `error: true`
 * so the UI can show a load failure rather than spinning forever.
 */
export function useSmilesDrawer() {
    const [state, setState] = useState<{
        ready: boolean
        error: boolean
        drawer: any
    }>({ready: false, error: false, drawer: null})

    useEffect(() => {
        if (typeof window === "undefined") return
        if ((window as any).SmilesDrawer) {
            setState({ready: true, error: false, drawer: (window as any).SmilesDrawer})
            return
        }

        const start = Date.now()
        const interval = window.setInterval(() => {
            if ((window as any).SmilesDrawer) {
                window.clearInterval(interval)
                setState({ready: true, error: false, drawer: (window as any).SmilesDrawer})
                return
            }
            if (Date.now() - start > 5000) {
                window.clearInterval(interval)
                setState({ready: false, error: true, drawer: null})
            }
        }, 50)

        return () => window.clearInterval(interval)
    }, [])

    return state
}
```

```tsx
import {useEffect, useRef, useState} from "react"
import {Button} from "@/components/ui/button"
import {Input} from "@/components/ui/input"
import {Card} from "@/components/ui/card"
import {Badge} from "@/components/ui/badge"

export default function HeroPlayground() {
    const [smiles, setSmiles] = useState(heroExamples[0].smiles)
    const [activeIdx, setActiveIdx] = useState(0)
    const svgRef = useRef<SVGSVGElement>(null)
    const {ready, error, drawer} = useSmilesDrawer()

    useEffect(() => {
        if (!ready || !drawer || !svgRef.current) return
        try {
            const svg = svgRef.current
            while (svg.firstChild) svg.removeChild(svg.firstChild)
            const opts = new drawer.SvgDrawer({
                width: 540,
                height: 360,
                bondLength: 18,
                fontSizeLarge: 6,
                padding: 12,

            drawer.parse(smiles, function (tree: any) {
                opts.draw(tree, svg, document.documentElement.classList.contains("dark") ? "dark" : "light")
            }, function (_err: any) {
                // ignore parse errors in hero — fallback to empty render
            })
        } catch (e) {
            // swallow render errors in hero — never break the landing
        }
    }, [smiles, ready, drawer])

    return (
        <Card className="relative overflow-hidden border-border/60 bg-card/50 backdrop-blur p-6 sm:p-8">
            <div className="flex flex-wrap items-center gap-2 mb-4">
                {heroExamples.map((ex, i) => (
                    <button
                        key={ex.smiles}
                        onClick={() => {
                            setSmiles(ex.smiles)
                            setActiveIdx(i)
                        }}
                        className={
                            "inline-flex items-center gap-2 rounded-full border px-3 py-1 text-xs font-medium transition-colors " +
                            (i === activeIdx
                                ? "border-foreground bg-foreground text-background"
                                : "border-border bg-background text-muted-foreground hover:text-foreground")
                        }
                    >
                        <span>{ex.name}</span>
                        <Badge variant="secondary" className="px-1.5 py-0 text-[10px] font-normal">
                            {ex.kind}
                        </Badge>
                    </button>
                ))}
            </div>
            <div className="flex gap-2 mb-4">
                <Input
                    value={smiles}
                    onChange={(e) => setSmiles(e.target.value)}
                    placeholder="Enter a SMILES string..."
                    className="font-mono text-sm"
                    spellCheck={false}
                    autoComplete="off"
                />
            </div>
            <div className="flex items-center justify-center min-h-[360px] rounded-md border border-border/60 bg-background/50">
                {error ? (
                    <div className="text-sm text-destructive">Failed to load SmilesDrawer</div>
                ) : !ready ? (
                    <div className="text-sm text-muted-foreground">Loading...</div>
                ) : (
                    <svg ref={svgRef} className="w-full max-w-[540px]" />
                )}
            </div>
        </Card>
```

```tsx
function BatchPanel({drawer, ready}: {drawer: any; ready: boolean}) {
    const [input, setInput] = useState("")
    const [results, setResults] = useState<Array<{smiles: string; svg: string; error: string | null}>>([])

    function handleRender() {
        if (!ready || !drawer) return
        const lines = input.split("\n").map((l) => l.trim()).filter((l) => l.length > 0)
        const out: typeof results = []
        for (const line of lines) {
            try {
                const tempSvg = document.createElementNS("http://www.w3.org/2000/svg", "svg")
                const svgDrawer = new drawer.SvgDrawer({
                    width: 400,
                    height: 320,
                    bondLength: 19,
                })
                let success = false
                drawer.parse(line, function (tree: any) {
                    svgDrawer.draw(tree, tempSvg, document.documentElement.classList.contains("dark") ? "dark" : "light")
                    success = true
                }, function () {
                    success = false
                })
                if (success) {
                    out.push({smiles: line, svg: new XMLSerializer().serializeToString(tempSvg), error: null})
                } else {
                    out.push({smiles: line, svg: "", error: "Parse failed"})
                }
            } catch (e: any) {
                out.push({smiles: line, svg: "", error: e?.message || "Render failed"})
            }
        }
        setResults(out)
        toast.success(`Rendered ${out.filter((r) => !r.error).length} of ${lines.length} molecules`)
    }

    function handleDownloadAll() {
        const successful = results.filter((r) => !r.error)
        for (let i = 0; i < successful.length; i++) {
            const blob = new Blob([successful[i].svg], {type: "image/svg+xml"})
            const url = URL.createObjectURL(blob)
            const a = document.createElement("a")
            a.href = url
            a.download = `molecule-${i + 1}.svg`
            a.click()
            URL.revokeObjectURL(url)
        }
        toast.success(`Downloaded ${successful.length} files`)
    }

    return (
        <div className="space-y-4 mt-4">
            <Card className="p-6">
                <Label className="text-sm font-medium text-foreground mb-2 block">SMILES (one per line)</Label>
                <textarea
                    value={input}
                    onChange={(e) => setInput(e.target.value)}
                    placeholder="CCO&#10;c1ccccc1&#10;CC(=O)O"
                    className="w-full h-40 px-3 py-2 rounded-md border border-input bg-background text-sm font-mono focus:outline-none focus:ring-2 focus:ring-ring"
                    spellCheck={false}
                />
                <div className="flex gap-2 mt-3">
                    <Button onClick={handleRender} disabled={!ready}>Render all</Button>
                    {results.length > 0 && (
                        <Button variant="outline" onClick={handleDownloadAll}>
                            <Download className="h-4 w-4 mr-2" />
                            Download all SVGs
                        </Button>
                    )}
                </div>
            </Card>

            {results.length > 0 && (
                <div className="grid grid-cols-2 md:grid-cols-3 lg:grid-cols-4 gap-4">
                    {results.map((r, i) => (
                        <Card key={i} className="p-3">
                            <div className="text-xs font-mono text-muted-foreground mb-2 truncate" title={r.smiles}>{r.smiles}</div>
                            {r.error ? (
                                <div className="text-xs text-destructive">{r.error}</div>
                            ) : (
                                <div className="bg-background rounded" dangerouslySetInnerHTML={{__html: r.svg}} />
                            )}
                        </Card>
                    ))}
                </div>
            )}
        </div>
    )
}
```

```tsx
        if (c) setCategory(c)
        if (t) setTheme(t)
        if (s) setSearch(s)
    }, [])

    // Apply filters to the DOM
    useEffect(() => {
        // Update query params (without reload)
        const params = new URLSearchParams(window.location.search)
        if (category === "all") params.delete("category"); else params.set("category", category)
        if (theme) params.set("theme", theme); else params.delete("theme")
        if (search) params.set("q", search); else params.delete("q")
        const query = params.toString()
        const newUrl = window.location.pathname + (query ? "?" + query : "")
        window.history.replaceState({}, "", newUrl)

        // Show/hide sections by category
        document.querySelectorAll<HTMLElement>("[data-example-section]").forEach((section) => {
            const key = section.dataset.exampleSection
            section.style.display = category === "all" || category === key ? "" : "none"
        })

        // Show/hide individual cards by search query
        const q = search.trim().toLowerCase()
        document.querySelectorAll<HTMLElement>("[data-example-card]").forEach((card) => {
            const name = (card.dataset.name || "").toLowerCase()
            const smiles = (card.dataset.smiles || "").toLowerCase()
            const matches = !q || name.includes(q) || smiles.includes(q)
            card.style.display = matches ? "" : "none"
        })

        // Dispatch a theme-change event the inline molecule scripts already listen for
        document.dispatchEvent(new CustomEvent("gallery-theme-change", {detail: {theme}}))
    }, [category, theme, search])

    return (
        <div className="rounded-lg border border-border bg-card p-4 sm:p-5 mb-8">
            <div className="grid gap-4 lg:grid-cols-[1fr_auto_auto] lg:items-end">
                <div>
                    <div className="text-[11px] font-medium uppercase tracking-[0.22em] text-muted-foreground mb-2">Category</div>
                    <Tabs value={category} onValueChange={setCategory}>
                        <TabsList className="flex-wrap h-auto">
                            <TabsTrigger value="all">All</TabsTrigger>
                            {categories.map((c) => (
                                <TabsTrigger key={c.key} value={c.key}>{c.title}</TabsTrigger>
                            ))}
                        </TabsList>
                    </Tabs>
                </div>
                <div>
                    <div className="text-[11px] font-medium uppercase tracking-[0.22em] text-muted-foreground mb-2">Render theme</div>
                    <Select value={theme} onValueChange={setTheme}>
                        <SelectTrigger className="min-w-[12rem]">
                            <SelectValue />
                        </SelectTrigger>
                        <SelectContent>
                            {themes.map((t) => (
                                <SelectItem key={t.value} value={t.value}>{t.label}</SelectItem>
                            ))}
                        </SelectContent>
                    </Select>
                </div>
                <div>
                    <div className="text-[11px] font-medium uppercase tracking-[0.22em] text-muted-foreground mb-2">Search</div>
                    <div className="relative">
                        <Search className="absolute left-3 top-1/2 -translate-y-1/2 h-4 w-4 text-muted-foreground" />
                        <Input
                            value={search}
                            onChange={(e) => setSearch(e.target.value)}
                            placeholder="Name or SMILES..."
                            className="pl-9 min-w-[14rem]"
                        />
                    </div>
                </div>
            </div>
        </div>
    )
}
```
