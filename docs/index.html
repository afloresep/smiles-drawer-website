Quick Start

Start with the default setup.

For most pages, the CDN script and SmilesDrawer.apply() are enough.

index.html 
<script src="https://unpkg.com/smiles-drawer@2/dist/smiles-drawer.min.js"></script>
<canvas data-smiles="Cn1cnc2c1C(=O)N(C(=O)N2C)C"></canvas>
<script>
    SmilesDrawer.apply({ width: 550, height: 450 });
</script>
API

Static pages

SmilesDrawer.apply()

Draw everything already in the page.

Interactive pages

new SmilesDrawer.SmiDrawer(...).draw()

Use this when the target or input changes.
